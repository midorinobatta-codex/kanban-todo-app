# FlowFocus Board design update

## 変更ファイル一覧
- `components/kanban-board.tsx`
- `reference_mockup.png`

## 今回の変更意図
- クイックキャプチャは常設のまま上部1段に集約
- `通常追加` と `定型` は必要な時だけ開く開閉式に変更
- モバイル / 中間幅で Board 本体が後ろに押し下がらないよう、Board を先に見せる順序に変更
- 左サイドは `検索 / 表示` と `履歴` 中心に整理

## 主なUI変更
1. クイックキャプチャを Board 上部の横長バーに移動
2. `通常追加` はトグルで開く補助パネル化
3. `定型` はトグルで開く補助パネル化
4. テンプレート一覧はスクロール内に収め、常時高さを取らない構成に変更
5. `error / notice` は上部操作エリア直下に表示
6. `xl` 未満では Board 側を先に表示し、サイド情報は後ろへ回す

## 反映手順
1. ZIP内の `components/kanban-board.tsx` を既存リポジトリの同名パスに上書き
2. 開発サーバーを再起動
3. Board 画面で表示確認

## 動作確認ポイント
- 初期表示で Board 本体が先に見える
- クイックキャプチャが上部1段で使える
- `通常追加` を押した時だけ既存作成フォームが開く
- `定型` を押した時だけテンプレート管理が開く
- `今日分を生成` が引き続き使える
- カンバン / 今日 / マトリクス / GTD 切替と D&D に影響がない
